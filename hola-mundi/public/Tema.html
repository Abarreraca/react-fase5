Selectores de tipo

Aplican estilos a todos los elementos de un tipo específico.

h1 { font-size: 28px; } p { line-height: 1.6; } a { color: #007bff; }

Selectores de clase

Permiten aplicar estilos a elementos con una clase específica.

.destacado { background-color: #ffeeba; padding: 10px; } .etiqueta { font-size: 12px; color: #777; }

Selectores de ID

Permiten aplicar estilos a elementos con un ID específico.

#encabezado { background-color: #333; color: #fff; padding: 20px; } #navegacion { background-color: #f8f9fa; }

Selectores de descendencia

Se utilizan para aplicar estilos a elementos anidados dentro de otros elementos.

ul li a { text-decoration: none; } .entrada .fecha { font-style: italic; }

Selectores de atributo

Permiten aplicar estilos a elementos que tienen un atributo específico.

[target="_blank"] { text-decoration: underline; } input[type="text"] { border: 1px solid #ccc; }

Selectores de grupo

Te permiten aplicar un conjunto de estilos a varios selectores al mismo tiempo.

h2, h3, h4 { color: #555; } .experiencia, .educacion, .habilidades { margin-bottom: 20px; }

Selectores de pseudo-clase y pseudo-elementos

Aplican estilos en estados específicos de elementos.

a:hover { color: #ff4500; } li::before { content: "➜ "; }
